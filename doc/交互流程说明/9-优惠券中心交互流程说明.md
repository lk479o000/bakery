# 优惠券中心（pages/coupon/coupon）交互流程说明

## 一、页面初始化流程

```
┌─────────────────────────────────────────────────────────────┐
│                     页面初始化流程                           │
└─────────────────────────────────────────────────────────────┘

onLoad
  │
  ├─> 1. 初始化页面数据
  │     - couponList: [] (优惠券列表)
  │     - loading: false (加载状态)
  │     - empty: false (空状态)
  │
  ├─> 2. 检查登录状态
  │     - 已登录：调用 loadCouponList() 加载优惠券
  │     - 未登录：显示未登录提示，引导用户登录
  │
  └─> 3. 页面渲染完成
```

### 数据结构设计

```typescript
interface PageData {
  // 优惠券相关
  couponList: Coupon[];
  loading: boolean;
  empty: boolean;
  
  // 登录状态
  isLoggedIn: boolean;
  
  // 操作状态
  receiving: boolean; // 防止重复领取
}
```

## 二、加载可领取优惠券列表流程

```
┌─────────────────────────────────────────────────────────────┐
│                   加载可领取优惠券列表流程                     │
└─────────────────────────────────────────────────────────────┘

loadCouponList()
  │
  ├─> 1. 显示加载状态
  │     this.setData({ loading: true, empty: false })
  │
  ├─> 2. 发起请求
  │     API: GET /api/coupon/list
  │     接口文档：services/coupon.ts:12-14
  │
  ├─> 3. 处理响应
  │     Success:
  │       - 更新 couponList
  │       - 检查是否为空数据
  │       - 设置 empty 状态
  │     Fail:
  │       - 显示错误提示
  │       - 保持原有数据不变
  │
  └─> 4. 更新页面状态
        this.setData({ 
          couponList: response,
          loading: false,
          empty: response.length === 0
        })

数据处理逻辑：
  - 后端返回的优惠券列表包含 received 字段，标识用户是否已领取
  - 页面显示时，已领取的优惠券按钮应显示为"已领取"状态
  - 未领取的优惠券按钮显示为"立即领取"状态
```

## 三、领取优惠券流程

```
┌─────────────────────────────────────────────────────────────┐
│                     领取优惠券流程                           │
└─────────────────────────────────────────────────────────────┘

用户点击"立即领取"按钮
  │
  ├─> 1. 检查登录状态
  │     - 未登录：跳转登录页
  │     - 已登录：继续流程
  │
  ├─> 2. 检查领取状态
  │     - 已领取：提示"已领取该优惠券"
  │     - 未领取：继续流程
  │
  ├─> 3. 显示领取中状态
  │     this.setData({ receiving: true })
  │     按钮显示为"领取中..."
  │
  ├─> 4. 发起领取请求
  │     API: POST /api/coupon/receive
  │     Params: { couponId: string }
  │     接口文档：services/coupon.ts:21-23
  │
  ├─> 5. 处理响应
  │     Success:
  │       - 显示成功提示
  │       - 更新该优惠券的 received 状态为 true
  │       - 更新按钮状态为"已领取"
  │     Fail:
  │       - 显示失败提示
  │       - 保持原有状态
  │
  └─> 6. 恢复可操作状态
        this.setData({ receiving: false })

异常处理：
  - 网络错误：提示"网络错误，请重试"
  - 服务端错误：根据错误码显示对应提示
  - 重复领取：提示"您已领取过该优惠券"
```

## 四、领取成功与失败提示流程

```
┌─────────────────────────────────────────────────────────────┐
│                   领取成功与失败提示流程                       │
└─────────────────────────────────────────────────────────────┘

领取成功：
  ├─> 1. 显示成功提示
  │     wx.showToast({
  │       title: '领取成功',
  │       icon: 'success',
  │       duration: 2000
  │     })
  │
  ├─> 2. 更新本地数据
  │     - 找到对应优惠券
  │     - 将 received 字段设置为 true
  │     - 更新页面显示
  │
  └─> 3. 可选：跳转到我的优惠券页面
        wx.navigateTo({
          url: '/pages/coupon/my-coupon'
        })

领取失败：
  ├─> 1. 显示失败提示
  │     wx.showToast({
  │       title: '领取失败，请重试',
  │       icon: 'none',
  │       duration: 2000
  │     })
  │
  └─> 2. 保持原状态
        - 按钮恢复为"立即领取"
        - 不更新页面数据

常见失败原因：
  - 优惠券已领完
  - 活动已结束
  - 用户领取次数限制
  - 网络异常
```

## 十、页面交互细节

### 1. 优惠券展示
- 每张优惠券显示名称、类型、面额、使用条件、有效期
- 已领取的优惠券显示"已领取"状态
- 未领取的优惠券显示"立即领取"按钮

### 2. 按钮状态管理
- 领取中：按钮显示"领取中..."，禁用点击
- 已领取：按钮显示"已领取"，禁用点击
- 可领取：按钮显示"立即领取"，可点击

### 3. 下拉刷新
- 支持下拉刷新功能
- 下拉时重新加载优惠券列表
- 显示刷新动画

### 4. 加载动画
- 页面初次加载时显示加载动画
- 领取优惠券时显示加载状态
- 加载完成后隐藏动画

### 5. 提示信息
- 领取成功：显示绿色对勾提示
- 领取失败：显示错误提示
- 网络异常：显示网络错误提示
- 未登录：显示登录引导提示

## 十一、代码实现建议

### 页面逻辑
```typescript
// 页面数据
Page({
  data: {
    couponList: [],
    loading: false,
    empty: false,
    receiving: false,
    isLoggedIn: false
  },

  // 页面加载
  onLoad() {
    this.checkLoginStatus();
  },

  // 页面显示
  onShow() {
    if (this.data.isLoggedIn) {
      this.loadCouponList();
    } else {
      this.checkLoginStatus();
    }
  },

  // 检查登录状态
  checkLoginStatus() {
    const token = wx.getStorageSync('token');
    const isLoggedIn = !!token;
    this.setData({ isLoggedIn });
    if (isLoggedIn) {
      this.loadCouponList();
    }
  },

  // 加载优惠券列表
  loadCouponList() {
    this.setData({ loading: true, empty: false });
    getCouponList()
      .then(res => {
        this.setData({
          couponList: res,
          loading: false,
          empty: res.length === 0
        });
      })
      .catch(err => {
        this.setData({ loading: false });
        wx.showToast({ title: '加载失败，请重试', icon: 'none' });
      });
  },

  // 领取优惠券
  receiveCoupon(e: { currentTarget: { dataset: { id: string } } }) {
    const couponId = e.currentTarget.dataset.id;
    
    if (!this.data.isLoggedIn) {
      wx.navigateTo({ url: '/pages/auth/login' });
      return;
    }

    if (this.data.receiving) return;

    const coupon = this.data.couponList.find(c => c.id === couponId);
    if (coupon?.received) {
      wx.showToast({ title: '已领取该优惠券', icon: 'none' });
      return;
    }

    this.setData({ receiving: true });
    receiveCoupon(couponId)
      .then(() => {
        wx.showToast({ title: '领取成功', icon: 'success' });
        // 更新优惠券状态
        const updatedList = this.data.couponList.map(c => 
          c.id === couponId ? { ...c, received: true } : c
        );
        this.setData({ couponList: updatedList });
      })
      .catch(err => {
        wx.showToast({ title: '领取失败，请重试', icon: 'none' });
      })
      .finally(() => {
        this.setData({ receiving: false });
      });
  },

  // 去登录
  goLogin() {
    wx.navigateTo({ url: '/pages/auth/login' });
  },

  // 去逛逛
  goShopping() {
    wx.switchTab({ url: '/pages/index/index' });
  },

  // 下拉刷新
  onPullDownRefresh() {
    this.loadCouponList();
    wx.stopPullDownRefresh();
  }
});
```

