## 三、数据库

| 技术 | 版本 | 说明 |
|------|------|------|
| MySQL | 8.0 | 主数据库 |
| Redis | 7.x | 缓存、会话存储 |

### MySQL配置建议
- 字符集: `utf8mb4`
- 排序规则: `utf8mb4_unicode_ci`
- 存储引擎: `InnoDB`

### Redis用途
- 用户会话存储
- 验证码缓存
- 热点数据缓存
- 限流计数

## 六、推荐技术栈组合

### 开发效率优先（推荐）
- **前端**: 微信小程序原生 + TypeScript
- **后端**: Node.js + Express + TypeScript
- **数据库**: MySQL 8.0 + Redis
- **部署**: 云服务器 / 云函数

**推荐理由**:
- 前后端都使用TypeScript，代码风格统一
- JavaScript生态丰富，开发效率高
- 适合中小型项目，快速迭代

## 七、环境要求

### 开发环境
- Node.js >= 18.0.0
- MySQL >= 8.0
- Redis >= 7.0

- 云服务器（2核4G起步）
