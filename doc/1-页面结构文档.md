## 一、TabBar 页面（底部导航）

```
│  [轮播图区域]                    │  ← 展示店铺宣传图
│                                 │
├─────────────────────────────────┤
│  [头像] 欢迎光临，请登录          │  ← 登录入口（未登录状态）
│          成为会员，享受更多权益   │
│                    [立即登录]    │
├─────────────────────────────────┤
│  HELLO                          │
│  欢迎光临 面包的日常              │
│                                 │
│    [自提图标]    [外卖图标]      │  ← 自提/外卖入口
│     自提          外卖           │
├─────────────────────────────────┤
│  [优惠券图标] 领券中心            │  ← 领券中心入口
│   COUPON COLLECTION CENTER      │
│                        [点击领取>]
├─────────────────────────────────┤
│  [充值图标] 会员充值              │  ← 会员充值入口
│   MEMBER RECHARGE               │
│                        [立即充值>]
├─────────────────────────────────┤
│                                 │
│      [面包师图标]                │  ← 品牌介绍
│       面包特点                   │
│    精选真材实料                  │
│   少科技添加，吃着更放心          │
│                                 │
│  [图标] 坚持选用新鲜顶级食材... [图标]
│                                 │
├─────────────────────────────────┤
│  [首页] [点单] [扫码] [订单] [我的] │  ← TabBar
└─────────────────────────────────┘
```

**页面元素**:
| 元素 | 类型 | 说明 |
|------|------|------|
| 轮播图 | swiper | 展示店铺宣传图片，支持自动轮播 |
| 登录卡片 | view | 未登录时显示，点击跳转登录页 |
| 自提按钮 | button | 跳转到点单页，默认选择自提 |
| 外卖按钮 | button | 跳转到点单页，默认选择外卖 |
| 领券中心 | cell | 跳转到优惠券列表页 |
| 会员充值 | cell | 跳转到充值页 |
| 品牌介绍 | view | 展示店铺特色介绍 |

**页面数据**:
- 轮播图列表
- 用户信息（判断是否登录）

**相关接口**:
- GET /api/banner/list - 获取轮播图
- GET /api/user/info - 获取用户信息

### 2. 点单页（pages/order/order）

**页面功能**: 商品点单页面，展示商品分类和商品列表，支持加入购物车

**页面布局**:
```
┌─────────────────────────────────┐
│  吉世面包                        │  ← 导航栏标题
├─────────────────────────────────┤
│  [自提/外卖切换]                 │  ← 配送方式选择
├─────────────────────────────────┤
│  分类列表 |                     │
│  ────────┤                     │
│  面包    │                     │
│  蛋糕    │    商品列表区域       │
│  饮品    │                     │
│  甜点    │                     │
│          │                     │
│          │                     │
├─────────────────────────────────┤
│  [购物车图标] ¥0.00  [去结算]    │  ← 底部购物车栏
├─────────────────────────────────┤
│  [首页] [点单] [扫码] [订单] [我的] │  ← TabBar
└─────────────────────────────────┘
```

**页面元素**:
| 元素 | 类型 | 说明 |
|------|------|------|
| 配送方式切换 | tabs | 自提/外卖切换 |
| 左侧分类栏 | sidebar | 商品分类列表，点击切换分类 |
| 商品列表 | list | 展示商品图片、名称、价格、加购按钮 |
| 购物车栏 | view | 显示购物车总价和数量 |
| 加购按钮 | button | 将商品加入购物车 |

**页面数据**:
- 商品分类列表
- 当前分类商品列表
- 购物车数量/金额

**相关接口**:
- GET /api/category/list - 获取分类列表
- GET /api/product/list - 获取商品列表
- POST /api/cart/add - 添加商品到购物车
