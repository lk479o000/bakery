# 接口文档

## 基础信息

- 基础URL: `https://api.example.com`
- 请求格式: JSON
- 响应格式: JSON
- 字符编码: UTF-8

## 响应规范

### 成功响应

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 错误响应

```json
{
  "code": 400,
  "message": "错误信息",
  "data": null
}
```

### 状态码说明

| 状态码 | 说明       |
| --- | -------- |
| 200 | 请求成功     |
| 400 | 请求参数错误   |
| 401 | 未授权，需要登录 |
| 403 | 禁止访问     |
| 404 | 资源不存在    |
| 500 | 服务器内部错误  |

## 1. 认证模块

### 1.1 微信登录

**接口地址**: `POST /api/auth/login`

**接口说明**: 微信小程序登录，获取用户openid并创建或更新用户信息

**请求参数**:

| 参数名  | 类型     | 必填 | 说明       |
| ---- | ------ | -- | -------- |
| code | string | 是  | 微信登录临时凭证 |

**请求示例**:

```json
{
  "code": "043xxxxxxxxxxxxxx"
}
```

**响应参数**:

| 参数名      | 类型     | 说明     |
| -------- | ------ | ------ |
| token    | string | 用户登录凭证 |
| userInfo | object | 用户信息   |

**响应示例**:

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIs...",
    "userInfo": {
      "id": "u123456789",
      "nickname": "微信用户",
      "avatar": "https://example.com/avatar.jpg",
      "phone": null,
      "points": 0,
      "balance": 0,
      "memberLevel": 1
    }
  }
}
```

| discountAmount | number | 优惠金额  |
| payAmount      | number | 实付金额  |

**响应示例**:

```json
{
```
