# 项目目录结构

## 项目概述

吉世面包小程序项目，包含微信小程序前端和 Node.js + Express 后端服务。

## 六、技术栈

### 前端
- 微信小程序原生框架
- TypeScript 4.x
- SCSS

### 后端
- Node.js 18+
- Express 4.x
- TypeScript
- JWT 鉴权
- MySQL 8.0
- Redis（可选）

### 部署
- Nginx + PM2
- HTTPS（Let's Encrypt）
- 腾讯云轻量服务器
